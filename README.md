👴👵 Senior Citizen Helper App

A simple, user-friendly web application designed to help senior citizens manage their daily health, routine, and safety.

---

🌟 Features

✔ Water Reminder 💧
Stay hydrated with timely water reminders

✔ Medicine Reminder 💊
Never miss medicines with alerts

✔ Daily Health Tips 📰
Get simple and useful health tips daily

✔ Memory Helper 🧠
Checklist for daily important tasks

✔ Daily Routine Reminder ⏰
Manage daily activities like meals, walk, sleep

✔ Missed Reminder Alerts 🔔
Get notified if you miss any reminder

✔ Cyber Safety / Scam Awareness 🛡️
Protect yourself from online frauds and scams
